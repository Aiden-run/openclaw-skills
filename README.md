你的 OpenClaw 可以帮你查黄金价格、刷 GitHub 热门、看 BOSS 直聘——这些技能一句命令装好，拿来就用。

## 一句话安装

把这句话复制给你的 Agent：

```
帮我从 https://github.com/Aiden-run/openclaw-skills/tree/main/skills 安装 skills
```

已经装过了？项目有更新时也会自动检测升级。
